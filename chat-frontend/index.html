Установить на «Домой»
  1. Нажмите кнопку «Поделиться» 􀈂 внизу Safari.
  2. Выберите «На экран “Домой”».
  3. Нажмите «Добавить» — появится иконка приложения.
  4. Откройте чат уже с иконки — тогда заработают уведомления.
